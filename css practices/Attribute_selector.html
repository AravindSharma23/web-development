Networking concepts

Client-server Model

In Client-server model ,the client sends the request to server over the internet.The Server send back the response to the client.In this model it uses http protocol for sending request and response.THe transmission of messages using http in air medium.

Peer to peer Model

In peer to peer model is also called p2p network.All the nodes are acts as both client and server.The transmission of messages are inside the network,It has atleast 12 clients are connected.

Webservices

In webservices are same as client server model.When an user enters a url on browser and this sends as a request to server and response to client as a result to it.
